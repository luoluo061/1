# 关键算法说明大纲

## 标准化
- 数值：Min-Max 缩放、对数变换。
- 类别：类别映射 + 校验、序数映射。
- 颜色：Hex 转 LAB（简化版）以支持色差。

## 特征编码
- One-hot：对离散特征展开维度。
- Ordinal：序数位置编码。
- Color LAB：三通道拼接。
- Scaler：单位范数、最大绝对值缩放。

## 权重与约束
- 载入 YAML 权重后执行 L1 归一化。
- 通过上下界约束防止配置错误。

## 相似度
- 余弦用于向量级比较。
- Jaccard 用于类别集合。
- Delta E 用于颜色距离，支持融合。

## 画像
- Schema 驱动的字段聚合。
- Narrative 模板组合数值阈值为描述语。
